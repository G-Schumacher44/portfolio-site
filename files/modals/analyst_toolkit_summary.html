Analyst Toolkit

A modular package that streamlines ETL, validation, and diagnostics for fast, consistent analytics — in Jupyter or via CLI.

System Summary

Centralizes reusable ETL, schema checks, duplicate/outlier detection, and clean exports into a configurable toolkit. Run interactively or as automated pipelines to speed setup, enforce hygiene, and standardize results across projects.
