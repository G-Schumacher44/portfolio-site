Executive Summary

A production-grade medallion lakehouse pipeline that turns raw e-commerce events into audited analytics marts. It combines contract-driven dbt models, Polars enrichment, and Airflow orchestration to enforce quality gates before data reaches BI.

  • 147 dbt tests across 16 validated tables with contract gating.
  • 3-stage validation (Bronze → Silver → Enriched) with quarantine handling.
  • 7x faster local enrichment versus warehouse Python models.
  • Audit JSON emitted per partition for SLA dashboards.

Architecture at a Glance

Bronze Layer

Raw Parquet ingestion from Google Cloud Storage with manifest validation and lineage metadata. Partitioned by ingestion date with full audit trails.

Base Silver (dbt-duckdb)

Type-safe transformations, integrity checks, and deduplication using DuckDB for fast local development. Enforces data contracts with primary key uniqueness and foreign key referential integrity.

Enriched Silver (Polars)

Business-aligned tables with precomputed metrics, cohort analysis, and behavioral features. Pure Polars transforms for performance and testability.

Gold Marts (dbt-bigquery)

Aggregated analytics tables optimized for BI and reporting in BigQuery. SQL-based transformations with partition-level incremental processing.
