Ecom Sales Data Generator

Realistic, relational e-commerce datasets for analytics practice and portfolio development.

System Summary

The engine behind the SQL Stories ecosystem. Generates configurable, realistic e-commerce datasets with customer behavior, product catalogs, orders, carts, and returns. Features seasonality simulation, customer lifecycle modeling, and controlled messiness levels for realistic analytics practice.

Tech Stack

Python Pandas SQLite YAML

Key Features

  • Relational integrity: Foreign key relationships between customers, orders, products, carts
  • Behavioral modeling: Customer lifecycle, purchase patterns, return behavior
  • Seasonality: Configurable seasonal trends and promotional periods
  • Messiness profiles: Controlled data quality issues for realistic cleaning practice
  • Volume control: Scale from hundreds to millions of rows
  • SQLite export: Ready-to-query database with indexed tables

Ecosystem Position

Provides the foundational data for the entire SQL Stories ecosystem, including the Skills Builder, Portfolio Demo case studies, and the Ecom Datalake Pipelines transformation layer.
